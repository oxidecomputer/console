# Colors

When using color, please double check that the color contrast follows the [WCAG guidelines](https://developer.mozilla.org/en-US/docs/Web/Accessibility/Understanding_WCAG/Perceivable/Color_contrast) for accessibility best practices.

## Usage

Colors are exposed directly on both the light theme and the dark theme in `props.theme.themeColors.COLOR_KEY`. Eventually the themes will abstract the colors so that you will not need to reference the theme colors directly.

## Accessories

Accessories are elements that can be placed inside the input area to provide added functionality.

### Icons

In order to centralize the styling of icons as accessories, you should use an `IconAccessory` instead of a plain `Icon`. `IconAccessory` shares the exact same API as `Icon`.

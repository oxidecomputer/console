# Shadows

There are 6 variants of box shadows, and one inner shadow.

## Usage

A `shadow` function is available to insert the appropriate `box-shadow` styles on your element.

```typescript
const Wrapper = styled.div`
  ${({ theme }) => theme.shadow()}
`
```

Passing no arguments to the `shadow` function returns the default shadow, there are also many other variants.

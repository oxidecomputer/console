# Colors

When using color, please double check that the color contrast follows the [WCAG guidelines](https://developer.mozilla.org/en-US/docs/Web/Accessibility/Understanding_WCAG/Perceivable/Color_contrast) for accessibility best practices.

## Usage

Colors will usually be applied through Tailwind classes like `text-green` or `bg-gray-400`. For cases not covered by Tailwind utility classes (like a one-off `box-shadow`), you can use the twin.macro `theme` import:

```js
import { styled, theme } from 'twin.macro'

const Input = styled.div`
  color: ${theme`colors.blue`};
`
```
